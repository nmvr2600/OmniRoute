## 🔌 ספקים

נהל חיבורי ספקי AI: ספקי OAuth (Claude Code, Codex, Gemini CLI), ספקי מפתח API (Groq, DeepSeek, OpenRouter), וספקים חינמיים (iFlow, Qwen, Kiro).

![Providers Dashboard](screenshots/01-providers.png)

---

## 🎨 שילובים

צור שילובי ניתוב מודלים עם 6 אסטרטגיות: מילוי ראשון, סיבוב סיבוב, כוח משתי בחירות, אקראי, פחות בשימוש ואופטימיזציה לעלות. כל משולבת שרשרת דגמים מרובים עם נפילה אוטומטית.

![Combos Dashboard](screenshots/02-combos.png)

---

## 📊 אנליטיקה

ניתוח שימוש מקיף עם צריכת אסימונים, הערכות עלויות, מפות חום של פעילות, תרשימי הפצה שבועיים ופירוטים לכל ספק.

![Analytics Dashboard](screenshots/03-analytics.png)

---

## 🏥 בריאות המערכת

ניטור בזמן אמת: זמן פעולה, זיכרון, גרסה, אחוזי חביון (p50/p95/p99), סטטיסטיקות מטמון ומצבי מפסק זרם של ספק.

![Health Dashboard](screenshots/04-health.png)

---

## 🔧 מגרש משחקים למתרגמים

ארבעה מצבים לאיתור באגים בתרגומי API: **Playground** (ממיר פורמטים), **Chat Tester** (בקשות חיות), **Test Bench** (בדיקות אצווה), ו**Live Monitor** (סטרימינג בזמן אמת).

![Translator Playground](screenshots/05-translator.png)

---

## ⚙️ הגדרות

הגדרות כלליות, אחסון מערכת, ניהול גיבוי (ייצוא/ייבוא מסד נתונים), מראה (מצב כהה/בהיר), אבטחה (כולל הגנת נקודות קצה API וחסימת ספקים מותאמים אישית), ניתוב, חוסן ותצורה מתקדמת.

![Settings Dashboard](screenshots/06-settings.png)

---

## 🔧 כלי CLI

תצורה בלחיצה אחת לכלי קידוד AI: Claude Code, Codex CLI, Gemini CLI, OpenClaw, Kilo Code ו-Antigravity.

![CLI Tools Dashboard](screenshots/07-cli-tools.png)

---

## 📝 יומני בקשות

רישום בקשות בזמן אמת עם סינון לפי ספק, דגם, חשבון ומפתח API. מציג קודי סטטוס, שימוש באסימונים, זמן אחזור ופרטי תגובה.

![Usage Logs](screenshots/08-usage.png)

## 🌐 נקודת קצה של ממשק API

נקודת הקצה המאוחדת של ה-API שלך עם פירוט יכולות: השלמות צ'אט, הטמעות, יצירת תמונות, דירוג מחדש, תמלול אודיו ומפתחות API רשומים.
